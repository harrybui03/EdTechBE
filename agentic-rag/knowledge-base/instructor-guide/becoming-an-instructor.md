# Becoming an Instructor

## Overview

To create and manage courses on the platform, you need to become an instructor first. This guide will walk you through the process.

## Step 1: Access the Become Instructor Page

1. Log in to your account
2. Navigate to the "Become an Instructor" page
3. You can find this option in the main navigation menu or user profile dropdown

## Step 2: Request Instructor Access

1. On the Become Instructor page, you'll see information about teaching on the platform
2. Click the "Get Started" button to request instructor access
3. The system will automatically assign you the `COURSE_CREATOR` role

## Step 3: Access Instructor Dashboard

After successfully becoming an instructor:

1. You'll be automatically redirected to the Instructor Dashboard
2. The dashboard is accessible via the "Instructor Dashboard" button in the top-right corner of the screen
3. You can also navigate directly to `/instructor` route

## Important Notes

- Once you have the `COURSE_CREATOR` role, you can create unlimited courses
- You can always access the instructor dashboard from the main navigation
- If you already have the instructor role, visiting the Become Instructor page will redirect you to the dashboard automatically

## Next Steps

After becoming an instructor, you can:
- Create your first course
- Manage existing courses
- View course analytics
- Set up payment information
