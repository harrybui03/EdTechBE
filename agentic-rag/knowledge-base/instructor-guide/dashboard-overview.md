# Instructor Dashboard Overview

## Overview

The Instructor Dashboard is your central hub for managing courses, viewing analytics, and accessing instructor tools. This guide provides an overview of dashboard features and navigation.

## Accessing the Dashboard

1. Log in to your account
2. Click "Instructor Dashboard" button in the top-right corner
3. Or navigate directly to `/instructor` route
4. You must have the `COURSE_CREATOR` role to access

## Dashboard Sections

### Courses Management

**View All Courses:**
- See list of all your courses
- Filter by status (Draft, Published)
- Search courses by title
- Quick access to course editing

**Course Status Indicators:**
- **Draft**: Course is being created/edited
- **Published**: Course is live and available to students

**Quick Actions:**
- Create new course
- Edit existing courses
- Preview courses
- View course analytics

### Course Creation

**Create New Course:**
1. Click "Create New Course" button
2. Follow the 3-step wizard:
   - Basic Information (title)
   - Tags and Labels
   - Time Information
3. Start building your course content

### Course Editing

Access comprehensive course editing tools:

**Plan Your Course:**
- Intended Learners: Define target audience
- Course Structure: Organize chapters and lessons

**Create Your Content:**
- Film & Edit: Manage video content
- Curriculum: Add and organize lessons
- Caption (optional): Add video captions
- Accessibility (optional): Configure accessibility features

**Publish Your Course:**
- Landing Page: Set up course presentation
- Pricing: Configure course pricing
- Promotions: Create discount codes

### Analytics and Statistics

**Course Performance:**
- Enrollment numbers
- Student progress
- Revenue (for paid courses)
- Student reviews and ratings

**Access Analytics:**
- Navigate to Statistics section
- View detailed course metrics
- Track student engagement
- Monitor course performance over time

### Batch Management

**Create and Manage Batches:**
- Organize students into learning groups
- Schedule live sessions
- Manage batch enrollments
- Track batch progress

## Navigation Tips

### Main Navigation

- **Courses**: Manage all your courses
- **Batches**: Create and manage learning batches
- **Statistics**: View analytics and performance
- **Payment**: Set up payment information

### Course-Specific Actions

From course list or detail page:
- **Edit**: Modify course content and settings
- **Preview**: See how course appears to students
- **Publish**: Make course available to students
- **Delete**: Remove course (use with caution)

## Troubleshooting

### Cannot Access Dashboard

If you can't access the dashboard:

1. Ensure you're logged in
2. Verify you have `COURSE_CREATOR` role
3. Try refreshing the page
4. Clear browser cache
5. Contact support if issue persists

### Courses Not Showing

If courses don't appear:

1. Check if you've created any courses
2. Verify you're on the correct account
3. Try refreshing the page
4. Check filters and search settings
5. Contact support if needed
